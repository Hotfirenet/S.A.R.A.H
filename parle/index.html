Le module parle permet de faire parler S.A.R.A.H en utilisant une simple requete http depuis votre box domotique.


l'utilisation se fait en appellant l'adresse de votre serveur nodejs.

Exemple: si votre ip est : 192.168.1.25

http://192.168.1.25:8080/sarah/parle?phrase=Bonjour Hotfirenet, comment vas tu ?
